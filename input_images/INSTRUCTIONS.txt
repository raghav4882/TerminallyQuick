Welcome to TerminallyQuick v3.0!

This is your default input folder for original images.

HOW TO USE:
1. Drop your images (PNG, JPEG, WEBP, HEIC, CR3, etc.) here.
2. Run the TerminallyQuick launcher for your OS (.command or .bat).
3. Follow the on-screen guides to optimize.

NEW IN v3.0:
- FAST TRACK: Select a saved Profile from the main menu to instantly process.
- SMART MODE: Let the app suggest optimal settings based on your images.
- PARALLEL SPEED: High-speed processing using all your CPU cores.

CUSTOM PATHS:
You can process images from ANY folder by choosing [C] Change Path. 
TIP: You can drag-and-drop any folder from your desktop into the 
terminal window when prompted for a path.
